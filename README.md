# Prodesk IT Landing Page

A responsive landing page built using **HTML** and **raw CSS** (Flexbox and Grid) as part of the **Week 1 Internship Task** at Prodesk IT.

This project focuses on mastering core frontend fundamentals like layout structuring, responsiveness, and clean UI without using any frameworks or libraries.

---

## Features

- Fully responsive design for mobile, tablet, and desktop
- Flexbox-based navigation bar
- Grid-based services section
- Clean and modern UI layout
- Cross-browser compatible

---

## Tech Stack

- HTML5  
- CSS3  
  - Flexbox  
  - CSS Grid
